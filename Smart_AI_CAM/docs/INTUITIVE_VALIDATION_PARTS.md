# 直覺式編程 — 驗收件型表

> **用途**：證明「簡單雙面件、少改動可上機」；每新增一類店內常見件就填一列。  
> **操作**：重新載入外掛 → 開啟該件 CAM Setup → 按面板 **一鍵直覺式編程** → 依下表勾選。

---

## 1. 單件驗收步驟（每件必做）

1. [ ] 設定頁已指定該材質之頂面／外輪廓／孔模板路徑  
2. [ ] **一鍵直覺式** 或 **檢查直覺式資格** → `eligible: true`  
3. [ ] 執行後 Fusion 刀路無 Error（Warning 可記備註）  
4. [ ] 與「資深手動套同一套模板」相比，**改動列數 ≤ 1**（或記下例外原因）  
5. [ ] 翻面／第二 Setup 若需要，**人工**完成（插件不代做）

---

**欄位說明**

- **資格通過**：一鍵直覺式前檢查或執行報告無「資格未通過」  
- **少改動**：執行後是否改孔模板下拉／2D 模板  
- **刀路 OK**：`generateToolpath` 後無 Error  

---

## 3. 通過標準（建議）

- 表中 **≥ 10 件** 且 **≥ 80%** 列同時滿足：資格通過 + 少改動 + 刀路 OK  
- 未通過件型：記入備註（例如有官方口袋、多階台），**不要**放寬白名單除非店規允許
