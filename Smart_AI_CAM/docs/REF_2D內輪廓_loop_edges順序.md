# REF：2D 輪廓工法 × 內輪廓（槽 inner loop）— loop_edges 順序

**狀態**：**暫定參考**。**不**取代 `docs/行為準則.md` §8.0～§8.4 之模板契約；變更實作仍須依 `docs/行為準則.md` §1。

**版號索引**：`docs/版本紀錄.md` 頂端（如 **V2.0310**）。

**一句話**：**`is_inner_loop` 為真 → 須反轉**；**外輪廓**則**不**反轉。內輪廓在 **B-rep** 語意下，**inner loop 的 coEdge 走向**對 **2D 輪廓工法**而言**永遠要反轉**才能切內側——這是**定義層**結論，不是單一零件外形的特例。實作上等價於 **`reversed(loop_edges)`**（或等價之鏈反轉）／或綁定後 **`ChainSelection.isReverted`**；**勿雙重反轉**。

## 0.0 Fusion 工法的幾何依據（總覽）

### 2D 工法

| 項目 | 內容 |
|------|------|
| **輸入** | **輪廓線**（**Chain／Loop**；實務上即鏈／封閉邊序，如 **`loop_edges`**） |
| **判斷** | **行進方向** + **compensation（補償／左／右）** |
| **結果** | **刀具沿線切削** |

### 3D 工法

| 項目 | 內容 |
|------|------|
| **輸入** | **面**（**BRepFace**） |
| **判斷** | **面的法向** + **材料側** |
| **結果** | **刀具掃過面** |

### 特徵帶入工法的對應（例）

| 特徵 | 幾何 | 對應工法（例） |
|------|------|----------------|
| 長條孔開口 | **inner loop**（2D 線） | **pocket2d**／**contour2d** |
| 圓孔壁 | **圓柱面**（3D 面） | **drilling**／**bore** |
| 平面頂面 | **平面**（3D 面） | **face**／**adaptive** |
| 外輪廓 | **outer loop**（2D 線） | **contour2d** |
| 斜面 | **斜平面**（3D 面） | **contour3d** |

### 結論（分工）

- **辨識器**：找出**正確的幾何**（該交鏈或交面、內輪廓或外輪廓）。
- **工法／工序**：**接受對應的幾何輸入**（2D 給線、3D 給面）。

**只要幾何對了，Fusion 自會依內建規則生成刀路**；實務除錯多半是「輸入幾何／線向與工法預期不一致」，而非另發明一套平行物理。

## 0. Fusion 設計思路（定案摘要）

1. **2D 輪廓工法的核心**  
   **刀具落點（補償側）** = 行進方向的 **left／right**。只有**行進方向**錯，就會切到錯側；過去大量分支本質上多在繞路回答「**要不要反轉**」。
2. **inner loop 的 `coEdge` 與 2D 輪廓工法**  
   **永遠需要反轉**才能切內側 ⇒ 來自 **B-rep 的定義**，不是零件長相的偶然性。
3. **Fusion 不做「完整 3D 上下離模」式的主觀推斷**  
   它以 **行進方向 + 實體不干涉** 為準；幾何／面資訊只需滿足**最小可驗證說明**即可驅動 2D 輪廓。（對話中曾以「**貓面**」描述此最小說明因；若實務上你指的是**模面／拔模面**等專有名詞，**結論不變**：仍以行進方向為一等公民。）
4. **複雜邏輯 ≠ 正確邏輯**  
   **Shoelace、圓心、面積、`host_face` 法向、XOR、原點** 等，多半是假象；**加一行 `reversed()`（或等價反轉）即可收斂**。

## 0.1 工法 × 輪廓：原則表（未來依據）

| 工法 | 輪廓類型 | 原則 |
|------|----------|------|
| **pocket2d** | **inner loop** | **反轉** |
| **contour2d** | **inner loop** | **反轉** |
| **chamfer2d** | **inner loop** | **反轉** |
| **pocket2d** | **outer loop** | **不反轉** |
| **contour2d** | **outer loop** | **不反轉** |

**規則**：**`is_inner_loop` 決定要不要反轉**。（槽語境下 **inner loop** 即內輪廓；**chamfer2d** 若為 **outer loop**，與上列 **outer** 列同理。）

## 2. 暫定規則（與 §0 對齊）

> **`is_inner_loop`** 為真 ⇒ **反轉 `loop_edges`（鏈／邊序）**；為假 ⇒ **不**反轉。以對齊手動 Fusion 之內／外輪廓側別。

實作可為建鏈前 **`reversed()`**，或綁定後 **`ChainSelection.isReverted`**（與 **`CAM2D_INNER_CONTOUR_FLIP_ALL_CHAINS_AFTER_BIND`** 同一語意空間）；**須避免雙重反轉**。
