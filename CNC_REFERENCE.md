## 1. 工件夾持系統（Workholding）

### 平口虎鉗（Vise）

| 項目 | 規範 |
|---|---|
| 適用工件 | 標準塊料、批量生產、規則外形 |
| 鉗口墊 | 軟銅片 / 樹脂塊（避免咬傷工件） |
| 夾持深度 | ≥ 3mm（避免加工時鬆脫） |
| 夾緊力 | 鋁料 5-10 kN，鋼料 10-20 kN |
| 注意 | 鉗口需先用百分表打平；活動鉗口磨耗每年校驗 |

### 真空台（Vacuum Plate）

| 項目 | 規範 |
|---|---|
| 適用工件 | 薄板（≥3mm AL / ≥2mm 鋼）、軟料、易夾傷 |
| 真空源 | -80 kPa 以上絕對真空，泵浦 ≥ 50 L/min |
| 密封圈 | O-ring 或矽膠條，每年換；表面有刮痕需研磨 |
| 加工限制 | Ap < 1mm 鋁，< 0.5mm 鋼；F 不超過 1500 mm/min |
| 注意 | 加工前用真空計確認；停電 / 真空斷必須有止擋兜底 |

### 磁性台（Magnetic Chuck）

| 項目 | 規範 |
|---|---|
| 適用工件 | 鋼鐵料、精磨件、批量生產 |
| 工件高度 | ≥ 5mm（薄料磁通量不足） |
| 磁力 | 永磁 ≥ 60 N/cm²，電磁 ≥ 80 N/cm² |
| 切削限制 | 大切削力時加機械止擋輔助 |
| 注意 | 鋁料 / 銅料 / 不鏽鋼磁性弱，不適用 |

### 治具（Fixture / Jig）

| 項目 | 規範 |
|---|---|
| 適用 | 量產異形件、複雜形狀 |
| 設計原則 | 3-2-1 定位（6 自由度）、預留刀具空間、易裝拆 |
| 材質 | 治具本體用 S45C/S50C，定位銷用 SKD11 (HRC60) |
| 公差 | 定位銷 H7/g6，定位面 ±0.01mm |

### 三爪卡盤 / 四爪卡盤

| 卡盤 | 用途 | 注意 |
|---|---|---|
| 三爪 | 圓棒、車削件 | 銑削時加配重；偏心夾不可用 |
| 四爪 | 異形、偏心 | 每爪獨立調整；可夾矩形 |

## 2. 找正與量測（Setup & Probing）

### 邊找正（Edge Finder）

| 工具 | 精度 | 適用 |
|---|---|---|
| 機械式 | ±0.005mm | 平面工件，X/Y 邊界找正 |
| 電子式 | ±0.002mm | 光潔平面，自動感應 |
| 光學式 | ±0.003mm | 透明工件、不接觸 |

操作：低速 S=500-1000 接觸工件 → 鍵入 X/Y 座標 → 退刀補半徑

### 圓心找（3D Probe）

- Renishaw OMP40 / OMP60 / OMP400 為主流
- 精度：±0.002mm
- 自動巡圓：`G65 P9810 X.. Y.. R..` (Renishaw 巨集)
- 可量孔深、孔徑、平面平行度、傾斜角

### Touch Off（觸接基準塊）

- 紙片法：傳統，憑手感，誤差 0.02-0.05mm
- 對刀儀（Z 軸）：精度 ±0.005mm，自動量刀長

### Z 高度設定方法比較

| 方法 | 精度 | 速度 | 適用 |
|---|---|---|---|
| 紙片 | 0.05mm | 慢 | 老師傅；不講究 |
| 對刀儀（手動） | 0.005mm | 中 | 標配 |
| 對刀儀（自動 G65） | 0.002mm | 快 | 量產 |
| 雷射對刀 | 0.001mm | 最快 | 高階機 |

## 3. 刀具補償與管理（Tool Compensation）

### 刀具長度補償（Tool Length Compensation）

```
G43 H## Z##    呼叫 H## 表中的長度補償，加到 Z
G44 H##        負向補償（少用）
G49            取消長度補償
```

**規則：**
- 換刀後第一行必須 `G43 H## Z##`，否則 Z 高度錯亂撞刀
- H## 表 = 刀具實際長度 - Z 工件零點到夾頭的距離
- 量測方式：對刀儀（自動）/ Z 軸基準塊（半自動）/ 紙片（手動）

### 刀具半徑補償（Tool Radius Compensation）

```
G40            取消半徑補償
G41 D##        左補（順銑用）
G42 D##        右補（逆銑用）
```

**注意：**
- 補償啟動需配合直線/圓弧進刀，不能直接在加工中間啟動
- 補償方向錯誤會導致過切或欠切
- D## 表 = 刀具實際半徑 + 預留量

### 刀具壽命管理

| 材質 | 鋁加工刀具壽命 | 鋼加工刀具壽命 |
|---|---|---|
| 高速鋼 (HSS) | 30-60 min | 5-15 min |
| 鎢鋼無塗層 | 60-180 min | 30-60 min |
| 鎢鋼 TiN | 90-240 min | 60-120 min |
| 鎢鋼 TiAlN/AlTiN | 120-300 min | 90-180 min |
| PCD（聚晶鑽石） | 1000+ min（鋁專用） | 不適用 |
| CBN（立方氮化硼） | 不適用 | 模具鋼 200+ min |

**監控指標：** 切屑顏色、聲音、震動、表面粗度、尺寸偏差

## 4. 量測與品管（QC）

### 在機量測（On-Machine Probing）

- 首件量：每批第 1 件量關鍵尺寸；超差停線
- 主動補償：量測結果回饋修正 G54 偏置
- 工具：`G65 P9814 X.. Y.. R..` (量孔徑)、`G65 P9821 D..` (量平面)

### 三次元量測（CMM）

| 機型 | 精度 | 用途 |
|---|---|---|
| Bridge CMM | 0.001mm | 大型件 |
| Gantry CMM | 0.002mm | 模具件 |
| Articulated arm | 0.020mm | 現場、大型 |

報告含：尺寸公差、形位公差（GD&T，平面度、平行度、垂直度、同軸度）

### 簡易現場量具

| 量具 | 精度 | 用途 |
|---|---|---|
| 卡尺 | 0.02mm | 粗量 |
| 千分尺 | 0.001mm | 軸徑、孔徑 |
| 內徑量錶 | 0.001mm | 深孔 |
| 高度規 | 0.01mm | 高度尺寸 |
| 量錶 | 0.001mm | 找平、跳動 |

### 表面粗度（Surface Roughness）

| Ra (μm) | 粗度等級 | 加工方式 |
|---|---|---|
| 12.5 | 粗銑 | 銑刀粗加工 |
| 6.3 | 半精 | 銑刀精加工 |
| 3.2 | 精銑 | 銑刀光刀 |
| 1.6 | 細銑 / 粗磨 | 球刀 / 磨削 |
| 0.8 | 細磨 | 平面磨 |
| 0.4 | 精磨 | 鏡面 |
| 0.1 | 拋光 | 手工 |

### 量具校驗

- 年度送公正單位（ISO 17025 認證）
- 內部校驗：每月用標準塊規檢查千分尺
- 損壞 / 跌落 / 異常 → 立即停用送校

## 5. 切削力學速查（Cutting Mechanics）

### 基本公式

```
v_c (切削速度, m/min)  = π × D × S / 1000
f_z (每齒進給, mm/tooth) = F / (S × Z)
h_m (平均切屑厚)        ≈ f_z × √(Ae/D)
MRR (材料移除率, cm³/min) = Ap × Ae × F / 1000
P_c (切削功率, kW)        = MRR × k_c / 60000   (k_c = 比切削力 N/mm²)
```

| 參數 | 符號 | 單位 |
|---|---|---|
| 主軸轉速 | S | rpm |
| 進給率 | F | mm/min |
| 刀徑 | D | mm |
| 刃數 | Z | - |
| 切深 | Ap | mm |
| 行距 | Ae | mm |

### 推薦切削速度範圍

| 材質 | v_c (m/min) | f_z (mm/tooth) | 比切削力 k_c (N/mm²) |
|---|---|---|---|
| AL6061 | 200-500 | 0.05-0.15 | 600-900 |
| AL7075 | 250-600 | 0.05-0.15 | 700-1000 |
| 黃銅 | 150-300 | 0.05-0.10 | 800-1200 |
| S45C/S50C | 80-200 | 0.03-0.10 | 2000-2800 |
| SKD11 | 50-150 | 0.02-0.08 | 2800-3500 |
| NAK80 | 80-180 | 0.03-0.10 | 2500-3000 |
| SUS304 | 50-150 | 0.03-0.08 | 2400-3000 |
| Ti-6Al-4V | 30-80 | 0.05-0.10 | 2700-3300 |

### 顫振（Chatter）防治

- 縮短刀具懸伸（L/D < 4）
- 降低切深，提高進給（fewer teeth in cut）
- 改變主軸轉速 ±10%
- 使用不等齒間距刀（variable helix）
- 提高機台剛性（夾具 / 工件 / 主軸）

## 6. 工序流程設計（Process Sequence）

### 標準流程（粗 → 精）

```
1. 面銑頂面         面銑刀 D50-D80, Ap 0.3-1.0
2. 開粗外輪廓       端銑刀 D8-D16, Ap 2-5 (鋁) / 1-3 (鋼)
3. 開粗內口袋       端銑刀 D6-D12, 螺旋下刀
4. 半精             同刀具，Stock 0.2-0.5mm
5. 精銑外輪廓       端銑刀 / 球刀，Ap 0.1-0.5
6. 精銑底面 / 內部  
7. 鑽中心孔         中心鑽 D2-D3
8. 鑽通孔 / 盲孔    依深徑比選 G81/G73/G83
9. 攻牙             G84 + M29 剛性
10. 倒角            倒角刀 D6-D10, 45°
```

### 流程設計原則

- **粗→精**：一鼓作氣完成粗加工，精銑前讓工件回溫消應力
- **上→下**：先頂面，後底面（避免變形）
- **外→內**：先外輪廓，後內口袋（穩定性）
- **硬料先**：模具鋼粗銑用銳利刀，後段用塗層刀
- **薄壁最後**：避免變形，最後精銑薄壁
- **倒角最後**：避免被後續工序撞擊

### 工序串接技巧

- 同把刀的工序連續做（減少換刀時間）
- T01 中心鑽 → T01 鑽 → T01 攻牙（同孔徑）
- 共用 H/D 表減少設定錯誤

## 7. 機台特性與選擇（Machine Selection）

### 立式 vs 臥式 vs 龍門

| 機型 | 行程 | 適用 | 剛性 |
|---|---|---|---|
| 立式 VMC | X 600-1500 | 中小件 | 中 |
| 臥式 HMC | X 800-2000 | 量產、4 軸 | 高 |
| 龍門 / Bridge | X 2000+ | 大型件、模具 | 最高 |
| 鑽攻機 | X 300-600 | 小件量產 | 低 |

### 主軸選擇

| 主軸類型 | 轉速 | 功率 | 適用 |
|---|---|---|---|
| BBT30 / BT30 | 12000-30000 | 5-15 kW | 鋁高速 |
| BBT40 / BT40 | 8000-15000 | 10-25 kW | 鋼料 |
| BBT50 / BT50 | 5000-10000 | 20-50 kW | 重切削 |
| HSK63A | 15000-30000 | 15-25 kW | 高速精密 |

### ATC（自動換刀器）

- 8-16 把 = 一般機；24-40 把 = 多工序機；80+ = 量產機
- 換刀時間：1-3 秒（Chip-to-chip）
- 注意：刀庫滿時需手動清不用刀具

## 8. 安全規範（Safety）

### 開機程序

1. 啟動冷卻系統（主軸冷卻 / 切削液 / 油霧）
2. 各軸歸零（Home）
3. 確認 G54-G59 工件原點
4. 檢查刀具庫，確認 H/D 表
5. 測試空跑（dry run）一次
6. 確認急停 / 限位 / 安全門

### 加工中監控

- 切屑形狀（理想：C 形 / 6 形短切屑）
- 聲音（變調 = 顫振或斷刀）
- 主軸負載（> 80% 應減負荷）
- 切削液流量（堵塞會燒刀）

### 異常處置

| 現象 | 可能原因 | 處置 |
|---|---|---|
| 顫振、噪音 | 刀具懸伸過長 / 進給過快 | 縮短懸伸 / 降 F |
| 斷刀 | 排屑不良 / 切深過大 | 啄鑽 / 降 Ap |
| 燒刀 | 切削液不足 / S 過高 | 補液 / 降 S |
| 工件鬆動 | 夾持不足 | 檢查 / 加強夾持 |
| 尺寸偏差 | 刀具磨耗 / 補償錯 | 換刀 / 重 D 表 |
| 撞機 | G43 漏寫 / G54 錯 | 立即急停 / 檢查 |

### 個人防護（PPE）

- 護目鏡（飛屑）
- 安全鞋（重物）
- 不戴手套（旋轉設備卡入）
- 長髮束起 / 無寬鬆衣物

## 9. 常見錯誤防範清單

```
[ ] 鑽完孔忘寫 G80 → 後續 G00 被當鑽孔
[ ] 攻牙忘 M29 / 同步進給 → 抓爆 / 斷牙
[ ] 鋁料用切削油 → 黏屑、表面粗糙；改乾切或氣冷
[ ] SKD11 用無塗層刀 → 刀具壽命短；改 AlTiN / TiAlN
[ ] 換刀忘 G43 + H## → Z 高度錯亂撞料
[ ] 細長刀粗銑大 Ap → 顫振、斷刀
[ ] 精銑前未消除應力 → 變形（鋁件熱處理回火）
[ ] 真空台薄板 Ap > 1mm → 工件浮動
[ ] M03 主軸未到達轉速就 G01 → 切削力衝擊
[ ] 銑削方向反了（順 vs 逆） → 表面粗糙 / 刀具壽命短
[ ] D 表錯誤（少加預留）→ 過切
[ ] 中心鑽深度不夠 → 後續鑽偏
```

## 10. 進階主題（後續可深化）

- **5 軸定位 vs 連續 5 軸** — Smart_AI_CAM 不支援，但 Fusion 原生有
- **Adaptive Clearing 動態切削** — 維持 h_m 恆定，刀具壽命提升 3-5x
- **HSM 高速加工** — 主軸 30000 rpm + Adaptive，適合鋁件量產
- **Trochoidal Milling 擺線銑削** — 槽銑的最佳策略
- **Plunge Milling 鑽銑** — 深口袋粗銑專用
- **Turn-Mill 車銑複合** — Smart_AI_CAM 不支援
- **EDM Wire / Sink** — Smart_AI_CAM 不支援
- **3D Printing / Additive** — Smart_AI_CAM 不支援
